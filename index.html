Step 02 — Know Your Baseline

Where are you right now?

Read the behavioral descriptions carefully — people often overestimate themselves and end up frustrated.

01

Complete Beginner

No real programming experience. Software feels like a black box.

✓ Can use a computer, understand file systems
✗ Cannot write a working function from scratch
✗ Cannot set up a dev environment alone
✗ Cannot explain what an API does in plain language
No prior code ~6–12 months to first role
02

Know Some Basics

Can read and write code. Stuck in tutorial hell — things break when you go off-script.

✓ Can write loops, functions, basic classes
✓ Completed at least one beginner tutorial or course
✗ Cannot build a complete project without following a tutorial step-by-step
✗ Cannot debug unfamiliar errors confidently
Basic syntax ✓ ~3–6 months to first role
03

Can Already Code

Independently builds complete apps. Weak on scale, system design, and production concerns.

✓ Built at least 2–3 complete projects without tutorials
✓ Comfortable with Git, a framework, and basic APIs
✗ Cannot design a system for 100K concurrent users
✗ Has not deployed anything to production with monitoring
Frameworks ✓ ~1–3 months to first role
